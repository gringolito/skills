---
name: address-pr-reviews
description: Address review comments on pull requests
argument-hint: The pull request number
---

Read all the review comments left on the pull request. Run a /grilling session, using the /domain-modeling skill, to clarify feedback understanding or fill in knowledge gaps before making changes.

Make sure every comment is addressed and that none are overlooked.

Delegate the implementation changes to any appropriate skill if available.

When you're done, reply to each review comment with a brief summary of how it was addressed.
